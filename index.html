Website Analysis Tool

This tool fetches basic SEO data from any publicly accessible URL. Enter a domain below (e.g. https://example.com) and click Analyze to view its meta title, description and word count. It’s designed to be extendable – additional data points can easily be added in future by modifying the server’s analysis logic or integrating third‑party APIs.
